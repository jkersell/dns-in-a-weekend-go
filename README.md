# DNS in a Weekend

This repository contains my implementation of Julia Evans' tutorial [DNS in a Weekend](https://implement-dns.wizardzines.com). The tutorial gives example code in Python but I wanted more challenge and an opportunity to practice writing in Go so I've written this implementation in Go.

The goal of the project is to implement a toy DNS resolver and to learn a bit about how DNS works in the process.

## Use of AI

I limited my use of AI to assisting in research to maximize learning.
